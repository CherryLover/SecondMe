### 后端 API

#### 话题接口
- [x] POST /api/topics - 创建话题
- [x] GET /api/topics - 话题列表
- [x] GET /api/topics/{id} - 获取话题
- [x] PATCH /api/topics/{id} - 更新标题

#### 消息接口
- [x] GET /api/topics/{id}/messages - 消息列表

- [x] GET /api/providers/{id}/models - 模型列表
