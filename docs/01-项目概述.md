# SecondMe - 项目概述

## 项目定位

SecondMe 是一个带有长期记忆能力的 AI 对话服务。通过向量数据库存储所有对话内容，实现跨话题的记忆检索，让 AI 随着使用时间的增长越来越"懂你"。

## 核心理念

- **记忆积累**：每次对话都会被记住
- **记忆检索**：新对话时，自动检索相关历史作为上下文
- **成长性**：使用越久，AI 对你的了解越深

## 目标用户

单用户个人使用（自己用）

## 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | Python + FastAPI | 简洁高效，AI 生态友好 |
| 前端 | HTML + CSS + JavaScript | 纯静态，无需编译 |
| 数据库 | SQLite | 零配置，文件型数据库 |
| 向量库 | ChromaDB | 本地向量数据库，底层 SQLite |
| AI 接口 | OpenAI SDK | 支持自定义 baseUrl，兼容 DeepSeek 等 |

## 项目结构

```
SecondMe/
├── server/          # 后端服务 (Python + FastAPI)
├── web/             # 前端页面 (HTML + JS)
└── docs/            # 项目文档
```

## 运行方式

本地单机运行，无需 Docker，无需云服务。
